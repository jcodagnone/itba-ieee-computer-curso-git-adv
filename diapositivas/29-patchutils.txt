│   lsdiff      - lists modified files in a patch                              │
│   rediff      - corrects hand-edited patches                                 │
│   fixcvsdiff  - corrects the output of 'cvs diff'                            │
│   splitdiff   - separates patches so that each new file alters a given file  │
